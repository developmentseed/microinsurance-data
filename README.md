### To create a new blog post:

1. Create a new `.md` file in the `posts` folder.
2. Use the following format
> Title: [TITLE]
>
> Author: [AUTHOR]
>
> Date: [DATE]
>
> %%%
>
> [Main text of the article]
   
3. Save & Commit
